Для тех кто заинтересован в "прокачке скилов"

27.04.2016

Сверстать небольшой Lending Page и опубликовать его на GitHub Page. Одностраничник должен быть определнной тематики. Например, "Сайт любителей цветов", "Геймерский сайт " или "Сайт интересных автомобилей". Верстать необходимо без использвоания сторонних библиотек, и фреймворков. Только HTML и CSS. В этом этапе задания не рекомендуется использвание JavaScript.
